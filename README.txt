1. git clone https://github.com/mxnazukx/aircompaniesmanagementsystem.git
2. Open cmd and go to aircompaniesmanagementsystem folder.
3. mvn install -DskipTests
4. docker-compose up --build -d
Service is started.

To work with service import postman-collection.json from postman-collection folder into postman. All enpoints are there.
